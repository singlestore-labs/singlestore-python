SingleStoreDB Python Client¶


The SingleStoreDB Python client is a Python DB-API compliant database connector for use with the SingleStore database. It also includes objects for interfacing with SingleStoreDB Cloud’s management API.

Warning

As of v0.5.0, the substition parameter has been changed from :1, :2, etc. for list parameters and :foo, :bar, etc. for dictionary parameters to %s and %(foo)s, %(bar)s, etc. respectively, to ease the transition from other MySQL Python packages.
